# Codex Step 1 — Directory Contents

## What Is This?

This directory contains everything needed to run **Step 1** of the ContextChat
Chrome Extension build in Codex (Claude Code). Feed these files to Codex and
it will scaffold the entire project.

## Files

| File | What It Is | When Codex Reads It |
|------|-----------|-------------------|
| `CONTEXT.md` | Project overview, architecture, tech stack, file structure | READ FIRST — gives Codex the big picture |
| `INSTRUCTIONS.md` | The actual Step 1 task with exact file contents to create | THE MAIN PROMPT — tells Codex what to do |
| `REFERENCE_CRXJS.md` | How to set up the CRXJS Vite plugin, known issues, fallback config | Reference during build config setup |
| `REFERENCE_TSCONFIG.md` | Required TypeScript settings, common type errors and fixes | Reference during tsconfig setup |
| `VERIFICATION.md` | 10 automated checks to verify Step 1 is complete | RUN AFTER — confirms everything works |

## How to Use

### Option A: Feed all files at once
1. Upload all 5 .md files to Codex
2. Tell Codex: "Read all the files I've uploaded. Start with CONTEXT.md, then follow INSTRUCTIONS.md to complete the task. After finishing, run the checks in VERIFICATION.md."

### Option B: Single combined prompt
1. Open INSTRUCTIONS.md
2. Copy the entire content
3. Paste into Codex as a single prompt
4. Codex has the CONTEXT embedded in the instructions already

### Option C: Step by step
1. First message: "Read this project context" → paste CONTEXT.md
2. Second message: "Now do this task" → paste INSTRUCTIONS.md
3. Third message: "Run these verification checks" → paste VERIFICATION.md
